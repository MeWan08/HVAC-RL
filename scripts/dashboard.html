What is this project?
🤖
Reinforcement Learning for HVAC
This project trains an AI agent to control the heating and cooling system of an office building. Instead of using fixed schedules, the agent learns the best setpoints by trying millions of scenarios and getting rewarded for saving energy while keeping occupants comfortable.

Think of it like teaching a thermostat to think — it reads 17 sensor values every 15 minutes and decides the optimal heating and cooling setpoints.
🏢
The Building & Simulation
The agent controls a 5-zone office building in Arizona (hot desert climate). The simulation uses EnergyPlus — the US Department of Energy's building physics engine used by real engineers worldwide.

Each training episode = 1 full year of simulation at 15-minute intervals, totalling 35,040 timesteps. Weather is randomised slightly each episode to prevent overfitting to a single year.
🧠
The SAC Algorithm
Soft Actor-Critic (SAC) is an off-policy RL algorithm designed for continuous action spaces — perfect for thermostat setpoints which aren't discrete buttons but a continuous range of temperatures.

SAC maintains an actor (picks actions) and a critic (evaluates them). The "soft" part adds entropy regularisation — encouraging exploration by rewarding the agent for being unpredictable, which prevents getting stuck in bad habits early in training.
🎯
Why does reward go negative?
The reward is always negative by design — the agent is penalised for both energy use and temperature discomfort. Closer to zero = better.

A mean reward of −0.22 means the agent is efficiently balancing both goals. A random agent gets around −0.50 or worse. The goal is to push the reward as close to 0 as possible.
Reward function
How the agent is scored every 15 minutes
LinearReward formula
rewardt = −( λ_energy × ω × HVAC_power_t + λ_temp × (1−ω) × |T_t − T_comfort| )
// negative always · optimal = 0 · agent must minimise both terms simultaneously
ω (energy weight) = 0.5
λ_energy = 0.0001
λ_temp = 1.0
Winter comfort = 20–23.5 °C
Summer comfort = 23–26 °C
ω=0 → comfort only · ω=1 → energy only
